# Air Crash Analysis Dashboard
## Project Overview

This project analyzes global air crash data to identify trends,
fatalities,countries with the highest number of crashes,and operators
involved in air accidents.

## obejectives
- Identify the year with the highest number of air crashes.
- Determine which countries experienced the most air crashes.
- Analyze air crash trends over time.
- Identify operators involved in the highest number of crashes.
- Examine how fatalities have changed over the years.

 
 ## Tools and Technologies
 - Python
 - Pandas
 - Matplotlip
 - Streamlit
 - Github

  ## Dataset
  The dataset contains historical air crash records including:
  - Year
  - Country/Region
  - Aircraft Manufacturer
  - Operator
  - Location
  - Fatalities
  - Number of people Aboard

   ## Key Findings
   - Certain countries recorded significantly more air crashes than others.
   - Air crash frequencies changed over time.
   - Some operators appeared more frequently in crash records.
   - Fatality trends varied across different years.
